# Variables Hub

Single-page hub for design variables: Figma tokens, normalized colors, palette, foundation tokens, and alignment with frontend CSS.

**Open:** [GitHub Pages link](#) — after you enable Pages, the site will be available at  
`https://<your-username>.github.io/color/`

## Local

Open `index.html` in a browser or run a local server:

```bash
python3 -m http.server 8000
```

Then open http://localhost:8000

## Contents

- **Colors** — semantic tokens, palette (Palette 26), comparison table (Figma / normalized / frontend)
- **Foundation** — radius, space, typography, effects (non-color variables)
- **Analytics** — shared parent summary, 3-way comparison, system strategy & optimization

Data sources: `colors26.json`, `pallette26.json`, embedded frontend tokens.
